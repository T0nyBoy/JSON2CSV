Hot it works

Paste an URL and press Convert to CSV. What you will get is either a message that you successfully converted the JSON data to CSV, and a prompt to download the CSV file, or an error message that the file was not converted either due to mistake on the URL or due to the structure of the JSON file.

Type of JSON data

The JSON data should be like:
a,b,c,d
a1,b1,c1,d1
a2,b2,c2,d2
a3,b3,c3,d3
It is basically a way to convert back to JSON data, whatever was exported from my other CSV to JSON script
